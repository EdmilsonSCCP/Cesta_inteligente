# Telas do Sistema

## 1. Dashboard

Exibir:

- Total gasto no mês
- Compras realizadas
- Produtos mais comprados
- Produtos próximos de acabar

---

## 2. Lista de Compras

Campos:

- Produto
- Quantidade
- Medida
- Categoria
- Supermercado
- Preço estimado

Ações:

- Adicionar item
- Remover item
- Marcar como comprado
- Finalizar compra

---

## 3. Produtos

Cadastro de produtos.

Campos:

- Nome
- Categoria
- Medida
- Marca
- Observação

---

## 4. Categorias

Cadastro de categorias.

Exemplos:

- Alimentação
- Limpeza
- Higiene
- Pets
- Café

---

## 5. Medidas

Cadastro de medidas.

Exemplos:

- Unidade
- Kg
- Litro
- Pacote
- Caixa

---

## 6. Supermercados

Cadastro dos supermercados.

Campos:

- Nome
- Endereço
- Observação

---

## 7. Histórico de Compras

Exibir:

- Data
- Mercado
- Valor total
- Produtos comprados

---

## 8. Histórico de Consumo

Exibir:

- Produtos mais comprados
- Frequência de compra
- Consumo médio mensal
- Preço médio
